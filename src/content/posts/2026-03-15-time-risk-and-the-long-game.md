I have become more interested in compounding than in intensity. Intensity is visible. Compounding is quieter, but it usually matters more.

That applies to money, learning, and relationships. Small decisions repeated over years can dominate dramatic decisions made in a hurry.

In investing, the simplest equation is still useful:

$$
\text{Future Value} = \text{Present Value} \times (1 + r)^n
$$

The hard part is not the formula. The hard part is staying rational long enough for the formula to work.

```js
const portfolio = [
  { name: "cash", weight: 0.2 },
  { name: "broad index", weight: 0.6 },
  { name: "conviction bets", weight: 0.2 },
];

const totalWeight = portfolio.reduce((sum, asset) => sum + asset.weight, 0);
```

I prefer a portfolio, and a life, that can survive mistakes. That means fewer heroic assumptions, more margin for error, and enough humility to keep learning after a good year.

The long game is not passive. It is disciplined attention applied over time.
